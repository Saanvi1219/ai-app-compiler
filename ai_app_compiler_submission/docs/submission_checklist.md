# Submission Checklist

## Required

- Live URL
- GitHub repository
- Loom video

## Before Submission

1. Push this project to GitHub.
2. Deploy using Render, Railway, Replit, or any simple Python hosting platform.
3. Record Loom video using docs/loom_script.md.
4. Submit:
   - Live URL
   - GitHub repo
   - Loom link

## Suggested Deployment

### Render

Start command:

```bash
uvicorn backend.main:app --host 0.0.0.0 --port $PORT
```

Build command:

```bash
pip install -r requirements.txt
```
